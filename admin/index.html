خطأ في إعداد Supabase

لوحة التحكم تحتاج رابط Supabase صحيح للعمل

📍 كيف تجد الرابط الصحيح؟

  1. اذهب إلى supabase.com/dashboard
  2. اختر مشروعك ثم اضغط Settings ← API
  3. انسخ Project URL — شكله:
    https://XXXXXXXXXXXX.supabase.co
  4. انسخ anon public key (يبدأ بـ eyJ أو sb_publishable_)
  5. الصق القيمتين في js/config.js
  6. أعد تحميل هذه الصفحة
