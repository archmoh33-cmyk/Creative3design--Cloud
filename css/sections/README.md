# css/sections/

ملفات CSS مخصّصة لأقسام محدّدة. الهدف: عزل styles كل قسم/كاروسيل/component جديد عن style.css و mobile.css المشتركَين، حتى لا يؤثّر تعديله على باقي الموقع.

راجع ../../PROTOCOL.md — قسم "قواعد كتابة الكود".

## قواعد التسمية

الصيغة: page-section.css

أمثلة:
- home-portfolio-carousel.css — كاروسيل البورتفوليو في الصفحة الرئيسية
- - home-calc-cards.css — بطاقات الحاسبات في الصفحة الرئيسية
  - - packages-calc-tabs.css — تابات الحاسبات في صفحة الباقات
    - - about-team-grid.css — شبكة الفريق في صفحة About
      - - blog-post-typography.css — typography المقالات في المدونة
       
        - تجنّب أسماء عامة مثل cards.css أو tabs.css — يجب أن يكون الاسم واضحاً للصفحة والقسم.
       
        - ## طريقة الاستخدام
       
        - 1) أنشئ ملف CSS جديد هنا بـ prefix فريد للـ classes
          2) 2) اربطه في الصفحة المعنيّة فقط باستخدام link rel=stylesheet
             3) 3) لا تربطه في صفحات أخرى — هذا هو سرّ العزل
               
                4) ## متى تستخدم هذا النهج؟
               
                5) استخدمه إذا:
                6) - تضيف قسماً جديداً كبيراً (أكثر من 5-10 قواعد CSS)
                   - - التعديل خاص بصفحة واحدة فقط
                     - - لا تريد المخاطرة بإضافة قواعد لـ style.css المشترك
                      
                       - لا تستخدمه إذا:
                       - - تعديل بسيط (1-3 قواعد) — ضعه في style داخل HTML
                         - - التعديل عام على كل الموقع — ضعه في style.css
                          
                           - هذا المجلد جزء من آلية التعديل الآمن — مفعّل منذ 2026-05-26.
